# 📅 캘린더 일정 관리 프로젝트

HTML, CSS, JavaScript로 만든 동적 캘린더 프로젝트입니다.

---

## 💡 주요 기능

- 현재 월, 요일별 날짜 자동 생성
- 이전/다음 달 이동 기능
- 오늘 날짜 하이라이트
- 날짜 클릭 이벤트 (확장 가능)

---

## 📁 파일 설명

| 파일명         | 설명 |
|----------------|------|
| index.html   | 캘린더 UI 및 네비게이션 |
| style.css    | 캘린더 디자인 적용 |
| script.js    | 날짜 생성 및 월별 이동 로직 포함 |

---

## 🧪 실행 방법

1. GitHub에서 프로젝트 다운로드 또는 클론

```
git clone https://github.com/taehvvan/calendar-app.git
cd calendar-app
open index.html
```
